tmux

tmux ls - list sessions

Ctrl + B :

    w - view all windows
    c - new window
    n - next window
    p - previous window
    , - rename window
    & - kill window

    % - vertical split
    " - horizontal split
    Arrow key - move between panes
    o - cycle panes
    q - show pane number
